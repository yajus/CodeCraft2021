## 程序文档

* Algorithm.java 进行服务的算法调度
* cost.java 服务器购买和维护的成本计算
* FileInOut.java 数据的输入和输出
* Service.java 数据输入所需要的服务器类
* VM.java 数据输入所需要的虚拟机类
* VMRequest.java 数据输出所需要的请求类
* VMPerDay.java 数据输入所需要的每天请求类
* Main.java 主程序
* range.java 排序 对于算法中新增服务器类型选择提供参考意见
* ServicesState.java 正在维护的服务器对象
* VMState.java 正在维护的虚拟机对象
